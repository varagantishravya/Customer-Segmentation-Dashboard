# Customer Segmentation Dashboard

## Project Overview
This project performs customer segmentation using K-Means clustering and visualizes customer behavior through an interactive Power BI dashboard.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Power BI

## Features
- Customer clustering using K-Means
- Income vs Spending analysis
- Interactive dashboard with slicers
- KPI cards and customer insights

## Outcome
Learned customer analytics, clustering techniques, dashboard design, and business insight generation.
